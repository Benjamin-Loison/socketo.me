Class BatchQueuePlugin

Queues requests and sends them in parallel when a flush event is recevied. You can call the flush() method on the plugin or emit a 'flush' event from the client on which the plugin is attached.

This plugin probably will not work well with plugins that implicitly send requests (ExponentialBackoffPlugin, CachePlugin) or CommandSets.
