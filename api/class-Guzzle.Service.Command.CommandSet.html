Class CommandSet

Container for sending sets of Guzzle\Service\Command\CommandInterface objects through Guzzle\Service\ClientInterface object.

Commands from different services using different clients can be sent in parallel if each command has an associated Guzzle\Service\ClientInterface before executing the set.
